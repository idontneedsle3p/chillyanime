# ChillyAnime — Anime Streaming Website

ChillyAnime is a web platform where users can watch anime online in a clean and simple interface.

🌐 Website: https://gochilly.fun

## Features
- Watch anime online
- Simple UI
- Fast loading
- Works on mobile

## Tech Stack
- FastAPI (Python)
- SQLite (aiosqlite)
- Tailwind CSS
- Jinja2

## Purpose
This project was created as a learning and portfolio project.
